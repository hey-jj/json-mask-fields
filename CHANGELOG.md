# Changelog

## [0.2.0] - 2026-07-07

### Changed
- Over-depth grouped selectors now drop the over-depth member instead of returning the parent subtree. (#20)
- Escaped stars inside key names now match literal `*` characters, so `a\*b` selects `a*b` instead of `a\*b`. (#21)
